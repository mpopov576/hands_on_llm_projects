# hands_on_llm_projects

A collection of Jupyter notebooks exploring LLM architectures and techniques hands-on, covering the full lifecycle from understanding how large language models work, to leveraging pretrained models, to fine-tuning them for specific tasks.

Built with PyTorch, Sentence Transformers, and LangChain.


## Installation

Clone the repository:

```bash
git clone https://github.com/mpopov576/hands_on_llm_projects.git
cd hands_on_llm_projects
```
